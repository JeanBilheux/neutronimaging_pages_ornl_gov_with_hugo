Data Analysis

How can I get help analyzing my data?

Just contact Jean Bilheux to discuss your needs.

By going over your experiment together, Jean will show you how to run the current tools and will develop customed python notebooks if needed.

What are those 'jupyter notebooks'?

The jupyter notebook developed by jupyter are an easy way to run python code using only a browser. By accessing our analysis computer, you won’t have anythign to install. Refer to our How To page to learn how to do that.

Where are my data and how can I access them?

The following tutorial will show you where are you data and how you can access them. Just go to How To > Access your data
